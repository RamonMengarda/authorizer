## Running Locally

To run this application locally, you'll need:

- Java 8
- Maven 4.0.0
- Spring Boot 2.7.5

### Running with Spring Boot

- On a terminal, navigate to the project's root folder
- Run `./mvnw spring-boot:run`
- The application will be available on port 8080
